# Mineradio v1.1.0 纯净安装版

这是 Mineradio 1.1.0 的纯净安装发布版。安装包从当前可信源码重新构建，旧 `dist`、旧安装包、旧备份包和旧 packaged build 都没有作为本次发布来源。

## 重要安全说明

- `v1.0.10` 及更早旧安装包不再建议继续安装或传播，请先隔离旧 `.exe` 安装包。
- 本次不提供从 `v1.0.10` 到 `v1.1.0` 的软件内本地更新，不提供快速补丁。
- 请手动下载 `Mineradio-1.1.0-Setup.exe`，卸载旧版本后进行纯净安装。
- 本次 Release 不上传 `latest.yml`，也不作为旧版软件内更新通道的 latest。

## 下载

- Windows 安装包：`Mineradio-1.1.0-Setup.exe`
- 校验文件：`Mineradio-1.1.0-SHA256SUMS.txt`

## 主要更新

- 将 `默认测试.json` 设置为首次启动默认用户存档，软件内默认视觉参数也同步使用这份快照。
- 新用户首次启动时，用户存档列表会自动预置「默认测试」。
- 默认视觉参数更新为：`emily` 预设、封面粒子分辨率 `1.55`、电影镜头强度 `0.5`、歌词发光 `0.28`、高亮色 `#fac900`、发光色 `#008aff`、UI/Home/视觉图标白色。
- 默认 3D 歌单架为静态常驻侧栏，侧向角度 `-15`，播客歌单默认关闭。
- 3D 歌单架新增「显示播客歌单」和「合并收藏歌单」开关。
- 高级设置新增后台策略和画质档位，直播后台保持开启后不会进入低占用暂停。
- 修复用户存档应用后切歌/跳歌回退到上一个视觉预设的问题。
- 修复 3D 歌单详情页动态/静态镜头绑定混乱、常驻触发区异常、详情页遮挡和滚轮高亮不同步问题。
- 修复详情页选歌时歌词透明度跳亮；详情页打开时歌词保持可读，但不遮挡中心高亮行。
- 修复全屏 DIY 悬浮入口遮挡视觉预设热键按钮，以及视觉控制台隐藏按钮残留的问题。
- 更新 Electron / electron-builder 依赖，并保留更新下载失败时的安全停住逻辑。

## 安装建议

1. 不要继续运行或转发旧安装包。
2. 先卸载旧版 Mineradio。
3. 隔离旧 `.exe` 安装包和旧打包目录。
4. 下载并安装 `Mineradio-1.1.0-Setup.exe`。
5. 安装后如需对照校验，请使用 Release 中提供的 SHA256 文件。

> 当前安装包暂未进行代码签名，Windows 可能显示安全提示。
